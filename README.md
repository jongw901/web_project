### 프로젝트 소개
###### 회원가입을 통해 글쓰기, 게시판 등을 활용할 수 있는 웹 페이지를 만든 프로젝트입니다.

### 기능
#### 1. 회원
- 회원가입
- 로그인
- 회원정보변경
- 회원 탈퇴
- 회원 쓴 글, 댓글 확인
#### 2. 게시판
- 글 작성
- 글 확인
- 댓글 작성
- 검색
